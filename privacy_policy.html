Privacy Policy - Firebase Storage Access API

Last Updated: May 19, 2025

1. Introduction

This Privacy Policy applies to the Firebase Storage Access API ("the API") that provides access to files stored in Firebase Storage for authorized users. The API is designed to be used with OpenAI's GPT platform to enable secure access to EEG datasets and related files.

2. Data Collection and Usage

The API only accesses files that are stored in your own Firebase Storage bucket. It does not collect, store, or process any personal data beyond what is necessary to authenticate and fulfill file access requests.

2.1 Information Collected

The API collects the following information:

  • API request metadata (timestamp, requested file paths, IP address)
  • Authentication information for verifying access rights

2.2 Purpose of Collection

This information is collected solely for:

  • Authenticating users and authorizing access to files
  • Providing the requested file information or content
  • Maintaining audit logs for security purposes
  • Improving the API functionality

3. Data Access and Control

The API follows the principle of least privilege and only accesses files that are explicitly requested by authorized users. All file access is subject to Firebase Storage security rules and authentication mechanisms.

You maintain full control over:

  • Which files are stored in your Firebase Storage bucket
  • Who can access the API
  • What operations can be performed on your files

4. Data Sharing

The API does not share any data with third parties except as necessary to fulfill its core functionality (i.e., retrieving files from Firebase Storage). File content is only shared with the authenticated requestor.

5. Security Measures

The API implements the following security measures:

  • Authentication using secure tokens
  • HTTPS encryption for all data in transit
  • Access logging for audit purposes
  • Adherence to Firebase security best practices

6. Data Retention

API access logs are retained for a maximum of 30 days for security and troubleshooting purposes. The API does not permanently store any file content - it only retrieves and serves files from your Firebase Storage bucket.

7. Changes to This Policy

This privacy policy may be updated from time to time to reflect changes in the API functionality or applicable regulations. Any significant changes will be communicated to API users.
